## Start project
- Clone this repo https://github.com/rodolphe37/magnetic-react-template.git
- Do in the console :
-    ```$ cd magnetic-react-template```
-    ```$ npm i``` or ```$ npm install ```
to install the dependencies and
-    ```$ npm start```
to start project for personalisation.

enjoy!!!
